# Online_Retail
Data analysis project using Power BI and SQL on Online Retail dataset.

The analysis was conducted using SQL for data extraction and transformation, and Power BI for data modeling and visualization.
The final output is an interactive dashboard designed for business decision-making.

**Tools & Technologies:**

SQL Server (SSMS): Data cleaning, transformation, and analysis
Power BI: Data modeling, DAX measures, and interactive dashboards

**Business Value:**

This dashboard helps businesses:
- Track sales performance over time
- Identify high-performing products and markets
- Understand customer value
- Optimize pricing strategies
